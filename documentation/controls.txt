------------------------------------------------------------------------
KEYBOARD SHORTCUTS

MAIN MENU SCREEN
Shortcut: Function (Alternative)
Q	: Exit program (ESC)
Space	: Start a new Pomodoro
NumPad+	: Increase window size (9)
NumPad-	: Decrease window size (0)

COUNTDOWN SCREEN
Shortcut: Function (Alternative)
Q	: Exit program
M	: Main Menu (ESC)
NumPad+	: Increase window size (9)
NumPad-	: Decrease window size (0)
